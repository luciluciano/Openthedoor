What is Nmap, really?

Nmap ("Network Mapper") is a free, open-source tool for network discovery and security auditing. It sends carefully crafted packets at target hosts, then analyzes the responses to answer four questions: which hosts are alive, which ports are open, what's running on them, and what OS the box is likely to be. Every scan you run is really one or more of the phases below, chained together.

05 · DEEP DIVE

The Nmap Scripting Engine (NSE)

NSE lets Nmap run Lua scripts against discovered hosts and ports — everything from banner grabbing to vulnerability checks. Scripts are organized into categories so you can run exactly the level of intrusiveness you intend, e.g. --script=safe or --script=default,vuln.
